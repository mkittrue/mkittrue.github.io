DOM基本概念

表示文档（如html和xml）和访问、操作构成文档的给雷元素的应用程序接口（API），DOM是指W3C定义的标准的文档对象模型，它以树结构表示html和xml文档，定义遍历了这个数和检查、修改树的节点的方法和属性。
接口主要有以下几类：
        核心Javascript语言参考（数据类型、运算符、基本语句、函数等）
        与数据类型相关的核心对象（String、Array、Math、Date等数据类型）
        浏览器对象（Window、location、History、Navigator）
        文档对象（Document、images、form）
还有种对象模型（BOM），提供访问浏览器各个功能组件，例如浏览器窗口本身、浏览历史的方法

例如，打开一个宽500高200的窗口，open('','_blank','width=500,height=200,menubar=no,toolbar=no,location=no,directories=no,status=no,scrollbars=yes,resizable=yes')
